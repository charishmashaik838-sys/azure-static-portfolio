# Azure Static Portfolio Website

## Project Overview

This project demonstrates how to deploy a static portfolio website using Microsoft Azure Storage Static Website Hosting.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Microsoft Azure Storage
- Azure Static Website Hosting
- GitHub

## Deployment Steps

1. Created Azure Storage Account.
2. Enabled Static Website Hosting.
3. Accessed the $web container.
4. Uploaded website files.
5. Accessed the website using the Azure Primary Endpoint URL.
